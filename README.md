# Smart Midday Meal Management System

## Description
Smart Midday Meal Management System is a Java Swing and MySQL desktop application developed to manage government school midday meals efficiently.

## Features
- Admin Login
- Cook Login
- Parent Login
- Daily Meal Management
- Stock Management
- Issue Management
- PDF Report Generation
- MySQL Database Connectivity

## Technologies Used
- Java
- Java Swing
- MySQL
- JDBC
- Apache NetBeans
